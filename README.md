# Task Manager API

Backend API for a Kanban-style task management application similar to Trello.

The project provides functionality for managing boards, columns, tasks, and team collaboration through invitations and role-based access control.

This repository contains only the backend part of the application.

---

## Features

- User authentication (JWT)
- Board management
- Columns inside boards
- Tasks inside columns
- User invitations to boards
- Role-based access control
- Error handling with custom exceptions
- DTO-based request/response mapping

---

## Tech Stack

- ASP.NET Core
- Entity Framework Core
- SQL Server
- JWT Authentication
- REST API
- Clean Architecture (CA)
